# 🦟 Dengue Disease Prediction System Using Machine Learning

## 📌 Overview  
This project presents an AI-based dengue detection system that predicts whether a patient is dengue positive or negative using blood test parameters. The system uses machine learning techniques along with a user-friendly web interface and data visualization dashboard.

## 📄 Abstract  
Dengue fever is one of the fastest spreading mosquito-borne diseases worldwide. Early detection is crucial for effective treatment. This system uses a Random Forest Classifier to predict dengue based on patient data such as hemoglobin, WBC count, platelet count, RBC count, and PDW.  

It integrates:
- Streamlit Web App for user interaction  
- Power BI Dashboard for visualization  

The system provides accurate, fast, and user-friendly dengue prediction.

## 🚀 Introduction  
Dengue is a viral disease transmitted by Aedes aegypti mosquitoes. Traditional diagnosis methods are time-consuming and may not be accessible in all areas.  

This project uses Machine Learning to analyze patient data and predict dengue quickly.

## ❗ Problem Statement  
Dengue diagnosis is often delayed due to limited healthcare resources and manual analysis.  
Solution: An automated ML-based system for quick prediction.
